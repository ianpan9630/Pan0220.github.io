Chack :
1. Add imput [ Jump ]
2. Set Animator : Jump State = 2 
3. Add State : [2] jump ( Base jump )
4. Set < Jump Parameters >
   JumpInPace 原地跳  \ Jump Forward 走路跳前  \ Jump Run 跑步狀態跳

注意 : 
角色地板探針 :
質新探針需往地板穿過一點點，可能會造成落地無法動彈

動作 :
1.原地跳 < Jump In pace >    \
2.向前跳 < Jump Forward   > \ Vertical : Greater 0.5
3.向前跳跑 < Jump Run   >     \ Vertical : Greater 2.5
